# Portfolio Website

This repository contains a static portfolio website built with HTML, CSS, and JavaScript.

## Repository Contents

- `index.html` — main portfolio page with sections for skills, experience, projects, education, and contact.
- `README.md` — repository documentation.

## Features

- Responsive layout for desktop and mobile.
- Animated hero section with typewriter effect.
- Skills, experience, projects, education, and contact sections.
- Simple navigation and mobile menu drawer.
- Custom animated background and scroll reveal effects.

## Usage

1. Open `index.html` in a web browser.
2. Or host the site using any static web server.

## Deployment

The site can be deployed on platforms such as GitHub Pages, Netlify, Vercel, or any static hosting service.

## Customization

- Update `index.html` to change page content and section text.
- Modify the inline CSS styles in `index.html` or extract them to a separate stylesheet.
- Adjust the JavaScript in `index.html` to change the typewriter, background animation, or scroll reveal behavior.
